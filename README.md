# 🏈 Football Spades - Coach's Edition

A football-themed Spades card game with premium UI and 3 difficulty levels.

## 🎮 Features
- Classic Spades gameplay with football scoring (7 points per trick)
- 3 bot difficulties (😈 Expert, 🤔 Medium, 😅 Easy)
- Premium leather and gold UI design
- Score history tracking
- Career stats
- 12 unique coach personalities
